# Theme Toggle Animation

## What does this do?

Creates a smooth animated theme toggle switch with sun and moon icons. The animation includes a sliding knob, rotation effect, and glow transition to provide an engaging user experience.

## How is it used?

```html
<input type="checkbox" id="toggle">
<label for="toggle" class="toggle">
    <div class="icon sun">☀️</div>
    <div class="icon moon">🌙</div>
    <div class="knob"></div>
</label>
```

## Why is it useful?

This animation provides a visually appealing and interactive toggle component that aligns with EaseMotion CSS's animation-first philosophy. It can be used in dashboards, portfolios, landing pages, and modern web applications to enhance user interaction and showcase smooth CSS animations.
