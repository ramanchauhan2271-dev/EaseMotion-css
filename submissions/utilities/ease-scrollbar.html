Standard Auto Scrollbar

This panel uses the default .ease-scrollbar-auto styling. The scrollbar width is standard (12px on webkit) but adopts the project's custom color theme variables automatically.

Scroll snapping tracks adapt to page flow, rendering thick bounding thumbs on desktop screens to allow simple grab navigation. In Firefox, it matches the native browser width scaling.

By defining standard styles once at the root level, developers ensure that standard pages look integrated with other controls. Simply style your elements with this utility to unify scroll tracks.
